# Neovim Guideline

## 移动光标

想要移动光标，可以通过摁下`h`、`j`、`k`、`l`。

> 或者通过小键盘方向键

- `h` (left)
- `j` (down)
- `k` (up)
- `l` (right)

## 打开文件

通过`Neovim`打开文件可以通过如下命令：

```bash
nvim FILENAME
```

## 退出`neovim`

想要退出`neovim`可以通过如下命令

首先确保自己在`normal`模式，如果不在`normal`模式，可以通过`ESC`进入。

- `:q!` -> 放弃所有更改并退出。
- `:wq` -> 保存所有更改并退出。(或者`:x`)

## 删除字符

我们可以通过键盘按键`x`来删除光标后的一个字符。

<Callout type="info">
必须在`Normal`模式下
</Callout>

## 添加文字

- `i`插入文字后进入`insert`模式，通过`ESC`退出至`normal`模式，插入位置在光标前。
- `a`插入文字后进入`insert`模式，通过`ESC`退出至`normal`模式，插入在光标位置后。

## 删除命令

通过`dw`可以删除一个单词。步骤如下：

- 通过`ESC`按键进入`normal`模式。
- 将光标移动到想删除的单词前。
- 通过`dw`删除掉这个单词。

## 更多删除命令

通过`d$`删除光标位置到本行末尾的所有字符。

- 通过`ESC`按键进入`normal`模式。
- 通过`d$`删除光标位置到结尾所有字符。

## `operators`和`motions`

很多改变文本的命令由`operators`和`motions`组成。

删除命令的格式如下：

```vim
d motion
```

其中：
- `d`是删除的`operator`
- `motion`是指`operator`将会如何执行

一个简短的`motion`列表如下：

- `w` - 从光标到下一个单词的开头
- `e` - 从光标到当前单词的结尾
- `$` - 从光标到本行末尾

## 在`motion`前使用计数器

在`motion`前输入一个数字代表重复这个`motion`多少次。

- 输入`2w`代表移动到光标的两个单词之后。
- 输入`3e`代表将光标移动都后三个单词的末尾
- 输入`0`代表移动到行首

## 在删除模式前使用`count`

格式：

```vim
d number motion
```

例如：

- `d2w`删除两个单词

## 删除行

通过`dd`可以删除一整行。

通过`ndd`可以删除n行。

比如：`2dd`可以删除两行。

## 回退操作

通过按键`u`可以回退上一步操作，`U`修复一整行。

通过`<c - r>`（Ctrl + R）可以前进一步操作。

## `Put`命令

`p`命令可以将删除的文本插入到光标前。

`P`命令可以将删除的文本插入到光标后。

## `Replace`命令

通关`r`命令可以将光标后的字符替换为我们想要的字符。

## `change`命令

通过`ce`命令可以删除当前光标到单词的末尾，并且进入`insert`模式。 

`c`命令的格式是：

```vim
c [number] motion
```
