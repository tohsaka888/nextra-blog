# List as abstract data type 列表的抽象数据类型

## 描述功能

- 存储指定的数字、元素或者指定的数据类型
- 能够在任意位置写入和修改元素
- 能够在任意位置读取元素

你可能回想，这不就是数组吗？

```cpp
// 存储int类型的元素
int a[10];

// 写入或修改元素
a[0] = 2;

// 读取元素
std::cout << a[0];
```

是的，数组是一种数据结构，可为我们提供此列表的实现。

现在，我们想要的列表具有更强大的功能。我们不再想要一个静态的列表，而是想要个动态的，所以，我们需要改一下它的功能。

## 更强大的功能列表

- 在列表为空时，长度为0
- 可以在现有的列表的任何位置插入元素
- 可以移除列表的某个元素
- 可以计算出列表的长度
- 可以在任意位置读写/修改列表
- 可以指定列表的数据类型
