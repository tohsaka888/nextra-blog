# Memory Units 存储单元

在开始讲述**内存**(*Memory*)之前，我们先来介绍一下`Register`。

## Register 寄存器

在本课程中，若没有特殊说明的情况下，`Register`默认指16位寄存器。

![img](https://files.catbox.moe/73bq3c.png)

- 读： 要读取寄存器的内容，只需要获取他的输出`out`。
- 写： 要将新数据`d`写入寄存器，只需要将`d`置于`in`输入管脚，并将`load`输入管脚置为1。这样，在下一个时钟周期内，寄存器被赋予了新的数据值，
他的输出变成了`d`，`load`置为0则保留上一次存贮的`d`。 

## RAM 随机存取存储器

可进行直接访问的记忆单元也成为RAM，是由n个w位寄存器组成的阵列。寄存器的个数（n）和每个寄存器的宽度（w）分别称为内存的
尺寸(*size*)和宽度(*width*)，我们将构建一组这样的存储体系，都是16位宽，但是大小不同：RAM8、RAM64、RAM512、RAM4K和RAM16K。
所有这些内存芯片都有相同的`API`。

![img](https://files.catbox.moe/029rnj.png)

> 注意：address的长度由存储大小决定

- 读： 要读取编号为`m`的寄存器，我们先将`m`置于`address`输入管脚，RAM找到编号`m`的寄存器并将它的输出发送到RAM的输出管脚。
- 写： 要将新数据`d`写入编号为`m`的寄存器，我们同样将`m`置于`address`的输入管脚，然后根据`load`位的输入来决定存储的值。
  - `load`为1，存储`in`
  - `load`为0，存储上一轮编号`m`的寄存器存储的数据
