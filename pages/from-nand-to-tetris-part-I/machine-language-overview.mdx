# Machine Language Overview 机器语言-概览

## 介绍

机器语言是一种约定的形式，用来对底层程序进行编码，从而形成一系列机器指令。
应用这些指令，程序员可以命令处理器执行算数和逻辑运算，在内存中执行存取操作，让数据在寄存器之间传递，
验证布尔表达式的值等等。与高级语言相比，机器语言的设计目的是能直接运行在指定的硬件平台上，并进行全面的操控。

### 概念

![img](https://files.catbox.moe/815a91.png)

简单来说：

- 开发者编写高级语言，例如：`C`、`Java`、`Python`......
- 编译器（*Compiler*）将高级语言翻译成机器语言
- CPU执行机器语言对应的指令

> 如果使用的是汇编语言，对应的编译器称为汇编器(*Assembler*)
